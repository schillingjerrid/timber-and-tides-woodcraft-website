# Timber & Tides Woodcraft site (Astro)

## Install
Use npm and the lockfile:
- npm ci

## Verify
Do NOT start a long-running dev server.
Use a static build as the verification step:
- npm run build

## Notes
- Node: 24.12.0
- This is a static Astro site deployed to GitHub Pages.
- Some sandboxes may not have reliable npm registry access. Use GitHub Actions CI as the source of truth.
