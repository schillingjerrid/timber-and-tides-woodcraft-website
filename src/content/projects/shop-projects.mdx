## Overview

Not every shop project ends up on display — some of the most useful builds are the jigs, sleds, and fixtures that make every other project better. This collection showcases some of those behind-the-scenes builds.

## Build Notes

- Shop jigs and fixtures built for specific tasks
- Materials selected for function over aesthetics
- Designs refined through actual use in the shop
- Some pieces adapted from well-known designs, others original
